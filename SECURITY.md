# Security Policy

## Supported Versions

We actively support and maintain the latest version

## Reporting a Vulnerability

If you discover a potential security issue or vulnerability, please **open a GitHub Issue** in this repository and describe:

- what you found (and how it might be exploited)
- which version you tested
- and any relevant details or examples

[Create a new Issue](https://github.com/krylosov-aa/context-async-sqlalchemy/issues/new)

Please avoid posting sensitive credentials or personal data in public issues.

---

## Response

We review new issues regularly and will:
1. confirm the report
2. investigate the problem
3. release a fixed version or publish mitigation steps if needed
